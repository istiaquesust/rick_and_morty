# rick_and_morty

1. Prerequisites

Make sure you have the Flutter SDK installed and configured on your machine.

2. Clone the Repository

Open your terminal and run:
  git clone https://github.com/istiaquesust/rick_and_morty.git
  cd rick_and_morty

3. Install Dependencies

Download all the necessary packages listed in the pubspec.yaml. Run following command in terminal:
  
  flutter pub get

5. Launch the App

Connect your device or start an emulator, then Run following command in terminal:

  flutter run
